## 개요

**TCMS**는 본사와 지점(또는 대리점) 단위 프로그램 운영이 필요한 기관에서 사용되는 통합 관리 솔루션입니다.

본사가 전체 고객 및 요금을 총괄하고, 각 지점은 지점별 고객을 직접 관리할 수 있도록 설계되어 본사와 지점 간의 **업무 효율**이 극대화됩니다.

지점별 프로그램을 본사에서 개별적으로 확인하거나 통계를 요청할 필요가 없어, **영업 및 데이터 공유** 측면에서도 매우 효과적입니다.

또한, CMS 자동이체 사용 시 본사의 단일 코드로 처리한 후, 지점별 비용을 구분 처리할 수 있어 **간편하면서도 저렴한 비용으로 효율적인 운영**이 가능합니다.

## 기능 안내

### 기본 기능

- 고객 관리
- 청구 및 수납 관리
- 상담 관리
- 계약서 파일 관리
- 요금 통계
- 월 서비스 관리

### 빌링 분야

- CMS 자동이체
- 지로(GIRO) 청구서
- 카드 자동이체
- 가상계좌

### 부가 기능

- 전자세금계산서
- 현금영수증
- 문자 발송
- 카카오 출금 전자서명
- 계좌 조회
- 실시간 계좌 승인

### 특수 기능

- 엑셀 일괄 업데이트
- 목록 필터링
- 로그 관리
- 지역별 권한 관리

※ 표기된 기능 외에도 다양한 편의 기능이 다수 제공됩니다.
