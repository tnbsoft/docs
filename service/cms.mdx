# CMS (Cash Management Service) 자동이체란?

CMS(Cash Management Service)는 은행 및 금융투자회사(증권사) 등 금융기관과 금융결제원이 공동으로 운영하는 서비스입니다. 이 서비스를 통해 이용기관은 고객 계좌에서 자금을 출금하거나 입금할 수 있어, 정기 또는 비정기 수납 업무를 보다 편리하고 신속하게 수행할 수 있습니다.

## 핵심 개념

- **자동이체**: 납부자가 직접 은행을 방문하지 않고도, 이용기관(기업)이 고객의 동의 하에 계좌에서 자금을 출금하여 지정된 계좌로 입금.
- **CMS 제도 개설 필수**: CMS 사용을 위해서는 CMS 제도 개설과 CMS 프로그램 사용이 반드시 필요합니다.

---

## ✅ CMS 자동이체 특징

### 🏢 이용기관 / 자금수납 기업

- **저비용 고효율**: 결제원에서 제공하는 안정적인 서비스를 최저 요금으로 이용 가능.
- **전 은행 대상 수납**: 국내 모든 은행 및 증권사 고객의 자금을 하나의 계좌로 수납 가능.
- **재출금 기능**: 미출금 시 재출금 요청이 가능하여 연체 관리가 용이하며 수납률 상승.
- **신속한 입금**: 고객 계좌 출금일(D) 다음 영업일(D\+1)에 자금이 입금 (6개월간 유예 가능).
- **자금 운용 효율화**: 정해진 날짜에 자금이 출금되어 체계적인 자금계획 수립 가능.

### 💼 납부자 / 고객

- **금융기관 자유 선택**: 국내 전 은행 및 증권사 계좌에서 출금 가능.
- **편리한 납부**: 정해진 날짜에 자동 출금되어 납부 방법 및 납기일에 대한 부담 감소.

## 🔧 CMS 자동이체 기능

### 1. 출금이체

렌털료, 구독료, 회비, 물품대금 등을 다수의 고객 계좌에서 인출하여 이용기관의 지정 계좌로 입금해주는 서비스입니다.

#### 출금 방식

| 종류                        | 설명                               |
| ------------------------- | -------------------------------- |
| **익일 출금**                 | 출금일 전일 요청 → 출금일 익일 결과 확인 (3일 처리) |
| **당일 출금**                 | 출금일 당일 요청 → 출금일 당일 결과 확인 (당일 처리) |
| **실시간 출금** _(2025년 준비 중)_ | 실시간 요청 및 처리 예정                   |

---

### 2. 입금이체

급여, 배당금, 적립금, 환불대금 등을 이용기관의 지정 계좌에서 다수의 고객 계좌로 입금해주는 서비스입니다.

## 📈 CMS 도입의 장점

| 항목         | 설명                               |
| ---------- | -------------------------------- |
| **납부 편의성** | 은행 방문 없이 자동이체로 간편하게 납부 가능        |
| **수납률 증가** | 최초 1회 신청 후 지속적인 출금 가능, 미수금 관리 용이 |
| **비용 절감**  | 대량 출금 자동화로 인건비 및 시간 절약           |
| **통계 관리**  | 전산화로 인한 정확한 통계 및 수납 현황 관리 가능     |
| **비대면 처리** | 고객과의 접촉 없이도 수납/지급 업무 처리 가능       |
