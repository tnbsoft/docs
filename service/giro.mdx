## 📄 지로(GIRO)란?

**지로 청구서**란 우리의 실생활에서 흔히 접하는 **요금청구서**입니다.\
**금융결제원**이 제도 관리를 수행하며, **기업의 각종 수납 요금**을 은행이 대행 수납하여 결과를 기업 계좌에 입금해주는 서비스입니다.

![지로프로세스.jpg](/images/지로프로세스.jpg)

---

## 🧾 장표 지로란?

지로는 이용기관이 **지로 장표에 고객의 납부 정보를 표시하여 송부**하고,\
고객은 **은행 창구나 전자 수납 매체**에서 요금을 납부합니다.

금융기관은 수납 내역을 종합해 기업에 수납 명세와 함께 자금을 입금해 줍니다.

![지로생성-2차.jpg](/images/지로생성-2차.jpg)

---

## 🧩 지로 장표의 종류

| 장표 구분       | 설명                            |
| ----------- | ----------------------------- |
| **OCR 장표**  | 납부자의 정보 및 금액을 사전에 지정해 발행하는 장표 |
| **MICR 장표** | 금액 미지정, 고객이 납부 금액을 자율적으로 입력   |

> ✅ 대부분 기업 수납 업무에는 **OCR 장표**가 활용됩니다.

---

## 🛠️ 지로 발행 프로세스

![지로발행프로세스.png](/images/지로발행프로세스.png)

1. 지로 이용기관 등록
2. 납부자 정보 등록
3. 지로 장표 발행 및 송부
4. 고객 납부 (은행 창구, 인터넷 등)
5. 금융결제원이 수납금 집계 및 결과 통보
6. 이용기관 계좌로 입금

## 📝 지로 개설 시 필요 서류

| 사업자 구분 | 필요서류                       |
| ------ | -------------------------- |
| **공통** | 사업자등록증 사본, 집금용 통장 사본       |
| **법인** | 법인 인감 증명서 원본, 법인 등기부 등본 원본 |
| **개인** | 대표자 인감 증명서 원본              |

> 📌 사업자등록증 발급일이 1년 이상일 경우, **사업자등록증명원**을 추가 제출해야 합니다.\
> 📌 **모든 인감도장 필수** 지참입니다.

## 📈 장점 vs 📉 단점

| 장점               | 단점                    |
| ---------------- | --------------------- |
| 상세한 납부내역 제공      | 발행/우편 등 아날로그 프로세스 필요  |
| 고객이 직접 납부 방식 선호  | 지로용지, 우편, 수수료 등 비용 부담 |
| 공공기관/보수적 기업에서 선호 | 친환경 흐름과 상충 가능성        |
